# SvgIcon

# 组件介绍

SvgIcon 是 SVG 图标组件，可以使用本地 SVG 文件，也可以使用网络中的图标资源。

  • 使用本地资源：将 SVG 文件保存到 /src/assets/icons/svg 目录下，SvgIcon 的 name 属性值使用文件名即可，不需要文件后缀。
  • 使用网络资源：使用网络资源图标中的 xx:xx 格式图标标识，作为SvgIcon 的 name 属性值即可 iconesopen in new window，iconifyopen in new window

提示

建议SvgIcon组件放到 el-icon 的插槽中使用，通过 el-icon 的属性设置图标的颜色及尺寸。

<!-- SvgIcon 需要引入 -->
<el-icon color="#fc5531" :size="16">
    <SvgIcon name="ep:home-filled"/>
</el-icon>
