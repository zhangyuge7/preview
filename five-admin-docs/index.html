FiveAdmin

一个开箱即用的后台管理系统框架

此项目已停止维护，请了解FiveAdminV2 open in new window 快速开始👉 在线预览 open in new window

💰开源

基于MIT协议开源，承诺无论个人学习、企业商用都可以免费使用或二次开发。（🚨切不可用于法律不允许，违法犯罪之事。）

🌈技术栈

基于JavaScript、Vue3、Vite4、Pinia、ElementPlus、Tailwindcss等技术栈开发。

🧰开箱即用

集成多语言支持、菜单权限、权限指令、黑暗主题、mock等常用功能，轻松接入后端。

🔐RBAC权限设计

基于RBAC的前后端权限管理方案，动态权限菜单及按钮级别的细粒度权限控制。
